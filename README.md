# POS Backend

NestJS backend service for POS management.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy environment variables and adjust values:
   ```bash
   cp .env.example.develop .env
   ```
3. Start development server:
   ```bash
   npm run dev
   ```

## Available Scripts

- `npm run dev`: seeds database and starts the server in watch mode.
- `npm run build`: builds the application into `dist/`.
- `npm run start`: runs the application once.
- `npm run lint`: runs ESLint with auto-fix.
